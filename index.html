About Me

Hi, Nama saya Ivan Kurniawan, saya adalah web developer yang berdomisili di Bekasi. Saat ini sedang belajar HTML di JDT 16

Perjalanan saya berawal dari ketertarikan pada teknologi komputer sejak usia remaja, ketika pertama kali saya bereksperimen melakukan reverse engineering pada sebuah game dengan Cheat Engine, lalu mengotomatiskan modifikasi memori tersebut melalui hotkey dan antarmuka Direct3D.

Seiring waktu, saya beralih ke Web Development, mempelajari PHP, JavaScript, React, Node.js, dan Express untuk membangun aplikasi front‑end dan back‑end. Saat ini, fokus saya tertuju pada pengembangan aplikasi dengan integrasi RESTful API, dan pengelolaan data produk secara real‑time.

Berbekal pengalaman magang dan proyek freelance, saya menerapkan prinsip clean code dan best practices.
