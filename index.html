About Me

Halo, Saya Ivan Kurniawan — Fresh Graduate Teknik Informatika Universitas Borobudur, dan seorang Software Developer. Berdomisili di Bekasi.

Perjalanan saya berawal dari ketertarikan pada teknologi komputer sejak usia remaja, ketika pertama kali saya bereksperimen membuat Cheat / Program illegal untuk curang pada sebuah game dengan Cheat Engine, lalu mengotomatiskan serta memodifikasi memori game tersebut melalui hotkey dan antarmuka Direct3D dengan framework custom dan ImGui dengan Bahasa Pemrograman C++.

Seiring waktu, saya beralih ke Web Development dengan mempelajari PHP, JavaScript, React, Node.js, dan Express untuk membangun aplikasi front-end dan back-end. Saat ini, fokus saya tertuju pada pengembangan aplikasi dengan integrasi RESTful API, dan pengelolaan data produk secara real-time. Berbekal pengalaman magang dan bootcamp, saya menerapkan prinsip clean code dan best practices.

Saya percaya bahwa kalau berani mencoba, dan konsisten terus dalam bereksperimen, maka hal yang dulu terlihat rumit akan menjadi hal yang biasa dilakukan. Karena setiap baris kode adalah langkah menuju kemampuan yang lebih besar.
